# Design — meertaligheid (spraak + UI)

- **Datum:** 2026-07-18
- **Status:** Goedgekeurd (chat)

## Doel

Twee onafhankelijke taalinstellingen:

1. **Spraakherkenning** — Whisper: `nl` / `en` / `de` (default `nl`)
2. **Interface** — labels via JSON-locales (default `nl`, fallback `nl`)

## Config

```json
"speech_language": "nl",
"ui_language": "nl"
```

## Bestanden

- `i18n.py` — `t(key)`, laden/wisselen UI-taal
- `locales/nl.json`, `en.json`, `de.json`
- Instellingen: twee dropdowns
- Live: spraak → `Opnamesessie.language`; UI → `i18n.set_language` + tray/settings verversen

## Buiten scope (v1)

OS-autodetect, gettext, meer talen, vertalen van code-comments.
